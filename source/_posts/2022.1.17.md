# 前缀和

## 公式

$a$是原数组，$pre$是前缀和数组，$b$是差分数组

**$pre_{i}=pre_{i-1}+a_{i}$**

差分：**$b_{i}=a_{i}-a_{i-1}$**

由上可知，**原数组是差分数组的前缀和数组，原数组是前缀和数组的差分数组**

## 优缺点

优点：查询只需要`O(1)`

缺点：动态数据不能使用前缀和

# 并查集

## 定义

可以合并和查找的集合

## 代码

### 查找

```C++
int fa[N];//记录自己的爸爸
int find(int x){
    return fa[x]==x?x:find(fa[x]);
}
```

### 合并

时间复杂度仅为`O(1)`

```C++
fa[find(a)]=find(b);
```

### 路径压缩

减少层级（在查找的过程中）

并查集复杂度：
压缩前：$O(log n)$
压缩后：$O(α(n))≈O(log log n)$

```C++
int fa[N];//记录自己的爸爸
int find(int x){
    return fa[x]==x?x:fa[x]=find(fa[x]);
}
```

### 初始化

必须将自己的爸爸指向自己

```C++
for(int i=1;i<=n;i++) fa[i]=i
```
