# 树状数组

维护、修改、查询区间信息

```cpp
int a[N],c[N];//c为树状数组

int lowbit(int x){
    return x&-x;
}

//查询
int Query(int x){
    int sum=0;
    for(int i=x;i;i-=lowbit(i)){
        sum+=c[i];
    }
    return sum;
}
//查询x~y区间的和：Query(y)-Query(x-1)
```
